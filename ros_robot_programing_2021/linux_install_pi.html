Linuxのインストール（Raspberry Pi）

Linuxはオープンソースのオペレーティングシステム（OS）です。

Raspberry PiではUbuntu Server 20.04 LTSを使います。


  • Raspberry Pi Imagerのインストール
  • ディスクイメージの書き込み
  • Linuxの設定
  • ネットワークの設定
  • ssh接続

Raspberry Pi Imagerのインストール

公式のイメージ書き込みツール「Raspberry Pi Imager」 を使ってSDカードにイメージを書き込みます。

詳しくは 公式HP を参照してください。

以下のコマンドを実行してRaspberry Pi Imagerを インストールします。 

    $ sudo apt install rpi-imager

Downloadsディレクトリにファイルが保存される。


7zipを起動し、Downloadフォルダに移動する。 ダウンロードしたファイルを選択して、「展開」ボタンを押す。

設定はデフォルトのままで、[OK]ボタンを押す。

解凍されるまで、しばらく待つ。

解凍された。

エクスプローラでダウンロードフォルダを確認する。


ディスクイメージの書き込み

Win32 Disk Imagerを起動する。

ディスクイメージを選択する。

「Device」に挿入したSDカードを選択して、 [Write]ボタンをクリックする。

[Yes]ボタンをクリックする。

インストール中。

[OK]ボタンをクリックする。


Linuxの設定

SDカードをRaspberry Piに挿入して電源を入れます。 ウィザードに従って設定してください。

今回使用するRaspberry Piは次のように設定してあります。

  1. Language: English

  2. Keyboard Layout: English(US)

  3. Time Zone: Tokyo

Langageは変更しないでください。 Keyboard Layoutは使用するキーボードに合わせて変更してください。 Time Zoneはタイ時間に合わせてください。
