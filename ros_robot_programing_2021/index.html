ROSロボットプログラミング

2021年度インターンシップ

使用する機器

  • PC(Ubuntu 20.04 LTS)
  • ロボット
    • Raspberry Pi 3 Model B
    • Arduino Uno
    • Zumo Shield
  • ジョイスティック
  • Webカメラ
  • USBケーブル ABタイプ
  • 単三乾電池 4本

ロボットはZumoを使います。 Zumoはコンパクトなロボット用プラットフォームキットで、 Raspberry PiとArduinoを使って制御します。

開発環境の構築

  • PC
    • Tera Termのインストール
    • VNC Viewerのインストール
  • Raspberry Pi
    • Linuxのインストール
    • ROSのインストール
    • Arduino IDEのインストール
    • SSHのインストール
    • VNC Serverのインストール
    • 自動ログインの設定
