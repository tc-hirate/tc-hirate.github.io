ROSでHello World

ROSで「Hello World」という文字列を表示するプログラムを 作成し、実行します。


  • 「Hello World」を表示する

「Hello World」を表示する

エディタで「hello_world.py」というファイルを作成します。 エディタは何でも構いません。VimやAtomなど好きなエディタを 使用してください。 ここでは、「pluma」という標準のエディタを使用しています。 

    $ pluma hello_world.py

以下のファイルを「hello_world.py」としてホーム直下に 保存してください。 

    import rospy
rospy.init_node('hello_world_node')
rospy.loginfo('Hello World')
rospy.spin()

それでは、作成したプログラムを実行しましょう。 最初に、roscoreを起動します。 

    $ roscore

もう一つTerminalを立ち上げて、次のコマンドを実行してください。

    $ python hello_world.py
[INFO] [1600821684.208287]: Hello World

「Hello World」と表示されました。 確認ができたら、[Ctrl-c]で終了してください。

それでは、一行づつ解説します。

    import rospy

rospyというPythonでROSを使うためのライブラリをインポート しています。 

    rospy.init_node('hello_world_node')

インポートしたrospyライブラリを使って、ROSのノードを作っています。 Nodeの名前は「hello_world_node」としました。 （他のノードと同じにならなければ、何でもいいです。） 

    rospy.loginfo('Hello World')

画面に「Hello World」という文字列を表示します。

    rospy.spin()

rospy.spinは無限ループで、[Ctrl-c]でプログラムが終了 するのを待っています。
