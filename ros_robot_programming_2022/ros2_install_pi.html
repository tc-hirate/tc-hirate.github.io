ROS2のインストール（Raspberry Pi）

Raspberry PiにROS2をインストールします。


  • ROS2のインストール
  • サンプルプログラムの実行

ROS2のインストール

公式HP の手順に従ってインストールします。
Binary packagesのDebian packagesを選択してください。

手順はPCと同じです。
Raspberry PiではGUI toolを使わないので、 desktop版ではなくbase版をインストールします。


サンプルプログラムの実行

インストールが正しく行われたか確認するために、 サンプルプログラムを実行します。
はじめに、setup fileを実行するコマンドを shellのstartup scriptに書いておきます。 

    $ echo "source /opt/ros/galactic/setup.bash" >> ~/.bashrc

IDを設定するコマンドも shellのstartup scriptに書いておきます。 IDはZumoの番号と同じにし、 Zumo-08を使っている場合は8となります。 

    $ echo "export ROS_DOMAIN_ID=8" >> ~/.bashrc

受講生のPCで次のコマンドを実行してください。

    $ ros2 run demo_nodes_cpp talker

受講生のRaspberry Piで次のコマンドを実行してください。

    $ ros2 topic list
/chatter
/parameter_events
/rosout
$ ros2 topic echo /chatter
