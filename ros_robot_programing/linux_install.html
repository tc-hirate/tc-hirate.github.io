Linuxのインストール

Linuxはオープンソースのオペレーティングシステム（OS）です。 今回は、Raspberry Piで使うことができるUbuntu MATE 18.04の 32bit版を使います。


  • Linuxのダウンロード
  • SDカードのフォーマット
  • ディスクイメージの書き込み
  • Linuxの設定

Linuxのダウンロード

Ubuntu MATE 18.04のディスクイメージを 公式HP からダウンロード。

「Or.... choose a hardware port:」の「Raspberry Pi 32-bit」 をクリック。

Downloadsディレクトリにファイルが保存される。


7zipを起動し、Downloadフォルダに移動する。 ダウンロードしたファイルを選択して、「展開」ボタンを押す。

設定はデフォルトのままで、[OK]ボタンを押す。

解凍されるまで、しばらく待つ。

解凍された。

エクスプローラでダウンロードフォルダを確認する。


SDカードのフォーマット

パッケージを認証する鍵の設定を行います。

    $ sudo apt-key adv --keyserver 'hkp://keyserver.ubuntu.com:80' --recv-key C1CF6E31E6BADE8868B172B4F42ED6FBAB17C654

ディスクイメージの書き込み

最初にソフトウェア・パッケージの情報を最新のものにします。

    $ sudo apt update

次にROSをインストールします。

    $ sudo apt install ros-melodic-desktop-full

途中で次のように表示されたら、そのまま[Enter]を押してください。

    Do you want to continue? [Y/n]

Linuxの設定

ROSのソフトウェアを使えるようにするには、 「ターミナル」を起動するたびに環境設定を行う必要があります。 次のコマンドを実行することで、「ターミナル」起動時に自動的に 環境設定が行われるようになります。 

    $ echo "source /opt/ros/melodic/setup.bash" >> ~/.bashrc
    $ source ~/.bashrc
