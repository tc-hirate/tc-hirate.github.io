VNC Serverのインストール

VNCとは、Virtual Network Computingの略で、 ネットワークを通じて接続された他のコンピューターの画面を 遠隔操作するソフトウェアのことです。

PCにVNC Serverを、Raspberry PiにVNC Viewerをインストールして、 PCからRaspberry PiをGUI（Graphical User Interface）で 操作できるようにします。


  • VNC Serverのインストール

VNC Serverのインストール

ソフトウェア・パッケージの情報を最新のものにする。

    $ sudo apt update

次のように表示されたら、パスワードを入力。

    [sudo] password for pi:

VNC Serverをインストール。

    $ sudo apt install vnc4server

次のように表示されたら、そのまま[Enter]を押す。

    Do you want to continue? [Y/n]

パスワードを設定。（zumo**、**は各自のZumoの番号）

    $ vncpasswd
Password:
Verify:

VNC Serverを起動。（ディスプレイ番号1として）

    $ vncserver :1

VNC Serverを停止。

    $ vncserver -kill :1

エディタでxstartupを編集。

    $ pluma ~/.vnc/xstartup

以下の2行を追記。

    ...
# exec /etc/x11/xinit/xinitrc
unset DBUS_SESSION_BUS_ADDRESS	6行目に追加する
...
x-window-manager &
exec /usr/bin/mate-session &	最終行に追加する

保存して、エディタを終了。
