使用する機器

  • PC(Windows 10)
  • ロボット
    • Raspberry Pi Model B
    • Arduino Leonard
    • Zumo Shield
  • ジョイスティック
  • Webカメラ
  • USBケーブル ABタイプ
  • 単三乾電池 4本

ロボットはZumoを使います。 Zumoはコンパクトなロボット用プラットフォームキットで、 Raspberry PiとArduinoを使って制御します。

事前準備

10/1に行います。

  • PC
    • Tera Termのインストール
    • VNC Viewerのインストール
  • ロボット
    • Linuxのインストール（実施済み）
    • ROSのインストール（実施済み）
    • Arduino IDEのインストール（実施済み）
    • IPアドレスの固定
  • リモート環境の設定
  • Linuxの基本操作
