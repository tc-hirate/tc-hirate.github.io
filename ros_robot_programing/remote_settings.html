リモート環境の設定

Ubuntuではaptというコマンドを使ってソフトウェアをインストールします。 UbuntuのバージョンとROSのバージョンには1対1の対応関係があり、 Ubuntu 18.04の場合はMelodicになります。

APT（Advanced Package Tool）とは、 Linuxでソフトウェアのインストールや管理、削除に用いられる パッケージ管理システムの一つです。 また、それを操作する「コマンド」のことをいいます。


  • SSHのインストール（Raspberry Pi）
  • 22番ポートの開放（Raspberry Pi）
  • 鍵の生成（Raspberry Pi）
  • Tera Termから接続（PC）
  • TeraTermからRaspberry Piを終了する
  • VNCサーバのインストール（Raspberry Pi）
  • VNCから接続（PC）

SSHのインストール（Raspberry Pi）

ソフトウェア・パッケージの情報を最新のものにする。

    $ sudo apt update

次のように表示されたら、パスワードを入力。

    [sudo] password for pi:

openssh-serverをインストール。

    $ sudo apt install openssh-server

次のように表示されたら、そのまま[Enter]を押す。

    Do you want to continue? [Y/n]

22番ポートの開放（Raspberry Pi）

Raspberry Pi側の22番ポートが塞がっていると接続できないので、 以下のコマンドを実行して、22番ポートを開放する。 

    $ sudo ufw allow 2
    $ sudo /etc/init.d/ssh restart
    $ sudo systemctl enable ssh

鍵の生成（Raspberry Pi）

SSHのホストキーが自動生成されていないので、 ホストキーを手動で生成する。 

    $ sudo ssh-keygen –A
    $ sudo /etc/init.d/ssh restart

Tera Termから接続（PC）

Tera Termを起動する。

「ホスト」と「IPアドレス」を入力して［OK］ボタンをクリック。

［続行］ボタンをクリック。

「ユーザ名」と「パスフレーズ」を入力して、［OK］ボタンをクリック。

接続が完了すると、次のような画面が表示される。


 

TeraTermからRaspberry Piを終了する

シャットダウン

    $ sudo shutdown -h now

再起動

    $ sudo shutdown -r now

VNCサーバのインストール（Raspberry Pi）

ソフトウェア・パッケージの情報を最新のものにする。

    $ sudo apt update

VNCサーバーをインストールする。

    $ sudo apt install vnc4server

次のように表示されたら、そのまま[Enter]を押す。

    Do you want to continue? [Y/n]

パスワードを設定する。（zumo**、**は各自のZumoの番号）

    $ vncpasswd
Password:
Verify:

VNCサーバーを起動する。（ディスプレイ番号1として）

    $ vncserver :1

VNCサーバーを停止する。

    $ vncserver -kill :1

エディタでxstartupを編集する。

    $ nano ~/.vnc/xstartup

以下の2行を追記する。

    ...
# exec /etc/x11/xinit/xinitrc	この次の行に追加する
unset DBUS_SESSION_BUS_ADDRESS
...
exec /usr/bin/mate-session &	最終行に追加する

Ctrl + Oで保存、Ctrl + Xでエディタを終了。


VNCから接続（PC）

デスクトップにある[VNC-Viewer]のショートカットをダブルクリックする。

[File]－[New connection]をクリックする。

「VNC Server」に172.16.21.**:1（**は各自のIPアドレスによる）と入力して、［OK］ボタンをクリックする。

接続先が追加された。

右クリック－[Connect]をクリックする。

次のウィンドウが出てきたら［Continue］ボタンをクリックする。

「Password」を入力して、［OK］ボタンをクリックする。

Windows上にubuntuのデスクトップが表示される。
