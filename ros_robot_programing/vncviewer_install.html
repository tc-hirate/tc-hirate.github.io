VNC Viewerのインストール

VNCとは、Virtual Network Computingの略で、 ネットワークを通じて接続された他のコンピューターの画面を 遠隔操作するソフトウェアのことです。

PCにVNC Viewerを、Raspberry PiにVNC Serverをインストールして、 PCからRaspberry PiをGUI（Graphical User Interface）で 操作できるようにします。


  • VNC Viewerのインストール
  • ショートカットの作成

VNC Viewerのインストール

VNC Viewerのインストーラ「VNC-Viewer-6.20.529-Windows.exe」を ダブルクリックする。

Englishを選択して、[OK]ボタンをクリックする。

[Next]ボタンをクリックする。

「I accept ...」にチェックを入れて、[Next]ボタンをクリックする。

[Next]ボタンをクリックする。

[Install]ボタンをクリックする。

インストール中。

[Finish]ボタンをクリックする。


ショートカットの作成

この操作は、ユーザーでログオンして行ってください。

「C:\Program Files\RealVNC\VNC Viewer」にある 「vncviewer.exe」を右クリックして、 [送る]－[デスクトップ（ショートカットを作成）]で ショートカットを作成する。
