Pythonの実行

ターミナルから「python3」と入力してインタラクティブシェルを起動する。 インタラクティブシェルを実行中は、先頭に「>>>」という記号が表示される。 

    pi@htc02:~$ python3
Python 3.6.9 (default, Jul 17 2020, 12:50:27)
[GCC 8.4.0] on linux
Type "help", "copyright", "credits" or "license" for more information.
>>>

「quit()」と入力してインタラクティブシェルを終了する。 

    >>> quit()
pi@htc02:~$

「python」と入力すると、Python2が起動するので注意。 今回は、Python3を使う。 

    pi@htc02:~$ python
Python 2.7.17 (default, Jul 20 2020, 15:37:01)
[GCC 7.5.0] on linux2
Type "help", "copyright", "credits" or "license" for more information.
>>>

数値と文字列

整数および浮動小数点を扱うことができる。 +、-、*、/で四則演算を行うことができ、 %で余りを、**で累乗を求めることができる。 

    >>> 0.5
0.5
>>> 1 +2
3
>>> 3 - 4
-1
>>> 5 * 6
30
>>> 7 / 8
0.875
>>> 10 % 9
1
>>> 3 ** 3
27

指数表記は次のように書く。

    >>> 1e-3
0.001
>>> 1e3
1000.0

文字列はシングルクォーテーションまたはダブルクォーテーションで囲む。 文字列の結合および繰り返しには+、*を使う。 

    >>> 'python'
'python'
>>> "python"
'python'
>>> 'python' + ' tutrial'
'python tutrial'
>>> 'python' * 3
'pythonpythonpython'

変数とコメント

数字や文字列、計算の結果を変数に代入することができる。

    >>> a = 1
>>> b = 2
>>> a + b
3
>>> c = a + b
>>> d = 'python'
>>> d * c
'pythonpythonpython'

変数に対する四則演算に関しては、 下記のような省略記法が用意されている。 

    >>> a = 1
>>> a += 2    # a = a + 2 と同じ意味
>>> a
3
>>> a -= 1    # a = a - 1 と同じ意味
>>> a
2
>>> a *= 3    # a = a * 3 と同じ意味
>>> a
6
>>> a /= 3    # a = a / 3 と同じ意味
>>> a
2.0

真偽値と比較演算子

真偽値を表すTrueおよびFalseという値がある。

    >>> 1 == 1   # 1と1は等しいか？
True
>>> 1 == 2   # 1と2は等しいか？
False

==以外にも!=、>、>=、<、<=という比較演算子がある。

    >>> 'python2' != 'python3' # python2とpython3は等しくないか？
True
>>> 2 > 3   # 2は3よりも大きいか？
False
>>> 2 >= 1  # 2は1以上か？
True
>>> a = 1
>>> b = 2
>>> a < b   # 変数どうしを比較することもできる。aはbより小さいか？
True
>>> b <= 2  # bは2以下か？
True

真偽値にはandおよびorという演算子を適用することができる。

    >>> a = 5
>>> 1 < a and a <10   #aは1より大きく、かつaは10より小さい
True
>>> 3 < a or a < 1    #aは3より小さい、またはaは1より小さい
True

リスト

複数の値をまとめて取り扱うことができるリストというデータ構造を 持っている。 他言語では、配列と呼ばれることもある。 

    >>> a = [1, 2, 3, 4, 5, 6]
>>> a
[1, 2, 3, 4, 5, 6]
>>> a[0]    # リストの要素にアクセス（インデックスは0から始まる）
1
>>> a[1]
2
>>> a[-1]   # インデックスにマイナスをつけると後ろから要素をたどる
6
>>> a[-2]
5
>>> a[1:3]  # 指定された範囲の値を取得（スライス）
[2, 3]
>>> a[2:]
[3, 4, 5, 6]
>>> a[:3]
[1, 2, 3]

制御構文

制御構文を使うことで、条件分岐や繰り返しをすることができる。 制御構文を使うとき、ブロックというまとまりでプログラムを書く。 Pythonの場合、インデントがブロックを表現している。 インデントは半角スペース4つを使う。

条件分岐にはif文を使う。 ifに続く式の真偽値がTrueならその下にあるコードブロックが実行される。 真偽値がFalseならelifの真偽値を見る。 そこもFalseならelseのブロックが実行される。 

    >>> a = 10
>>> if a % 3 == 0:
...     print('Dividable by 3')
... elif a % 5 == 0:
...     print('Dividable by 5')
... else:
...     print('Not Dividable by 3 or 5')
...
Dividable by 5

繰り返し処理にはfor文を使う。 forにリストを渡すことで、そのリストの中身を１つずつ取り出して 繰り返し処理させることができる。 

    >>> a = [1, 2, 3, 4, 5, 6]
>>> for i in a:
...    print(i)
...
1
2
3
4
5
6

繰り返し処理にwhile文を使うこともできる。 whileに続く式の真偽値がTrueの間はずっと繰り返し処理をする。 

    >>> a = 1
>>> while a <= 5:
...     print(a)
...     a += 1
...
1
2
3
4
5

関数

処理のまとまりを関数として定義することができる。 関数の定義にはdefを使い、その下にあるコードブロックが 関数の中身として定義される。 制御構文と同じようにインデントがコードブロックを意味する。 

    >>> def hello_python():
...     print('Hello Python')
...
>>> hello_python()
Hello Python
>>>
>>> def sum(a, b):
...     return a + b
...
>>> sum(1, 2)
3
